# expense-analysis-python
# 経費データ分析（Python）

## 概要
建設会社の経理業務を想定し、
現場別・月別の経費データをPythonで集計・可視化した分析です。

## 背景
現場から提出される経費データとシステムデータの照合作業が多く、
確認・集計を効率化するために本分析を作成しました。

## 使用技術
- Python
- pandas
- matplotlib
- Google Colab

## 内容
- データ読み込み
- データ前処理
- 現場別・月別集計
- 要確認データの抽出
- グラフによる可視化

## 補足
本分析は実務を想定したポートフォリオです。
